## Key decisions made
- Decision: Use **Cinzel** for the hero title and nav wordmark.
  Why: It's a Trajan-style inscriptional capital — the closest common web font to the engraved-serif lettering in the logo crest. Since the logo is an image, this is a visual match rather than the exact font.
- Decision: Keep the new logo as a transparent **PNG** (`logo.png`) rather than overwriting `logo.jpg`.
  Why: Transparent background sits cleanly on the dark nav/hero/footer; using a new filename also avoids cache issues with the old asset.
